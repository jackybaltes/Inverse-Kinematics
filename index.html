Problem Definition

Forward Kinematics

From joint coordinates to coordinates $$ θ_1, θ_2, ..., θ_i $$ of the end effector in the inertial frame

Inverse Kinematics

From target coordinates $$ x_e, y_e, z_e, ...$$ of end effector in the inertial frame to joint coordinates $$ ( θ_1, θ_2, ... θ_n ) $$
label slide_00004_frag_00001: show JB at left JB """Forward kinematics is useful in some applications, since we can find the coordinates of the end effector in the inertial frame given the joint coordinates""" JB """But more often we have to solve the inverse problem: given target coordinates for the end effector, find the joint coordinates"""

General Formulation

Target is a transformation matrix for forward kinematics
\[ H = \left[ \begin{array}{cc} R_{3x3} & o_{3x1} \\ 0 & 1 \\ \end{array} \right] \]

Find the compound transformation matrix \[ T_{n}^{0}(q_1, q_2, .... q_n) = H \]

\[ T_{n}^{0}(q_1, q_2, .... q_n) = A_1(q_1) A_2(q_2) A_3(q_3) ... A_n(q_n) \]

So we must find the joint variables \( q_1, q_2, ... q_n \) such that \[ A_1(q_1) A_2(q_2) A_3(q_3) ... A_n(q_n) = H \]

Even on simple robots, we often have infinite solutions

Two Link Solution

The target location for the hip is ( 0.13, 0.51 )

Convert it into polar coordinates with \[ \begin{array}{l} l_3 = \sqrt{ dx^2 + dy^2 } \\ \alpha = atan2( dy, dx ) \end{array} \]

l3 = 0.52, alpha = 75.98

Cosine Law

\[ l_3^2 = l_1^2 + l_2^2 - 2 l_1 l_2 \cos(\alpha_2) \]

We can then calculate the angle \(α_2\), since \(l_1\),\(l_2\) are all known

We can then calculate the angle of the knee

\[ \begin{array}{ccc} l_3^2 - l_1^2 - l_2^2 & = & - 2 l_1 l_2 \cos( \alpha_2 ) \\ \cos( \alpha_2 ) = \frac{ l_3^2 - l_1^2 - l_2^2 }{-2 l_1 l_2 } \\ \cos( \alpha_2 ) = \frac{ l_1^2 + l_2^2 - l_3^2}{2 l_1 l_2 } \\ \alpha_2 = \arccos( \frac{ l_1^2 + l_2^2 - l_3^2}{2 l_1 l_2 } ) \\ \end{array} \]

Inverse Kinematics Two Link System

Since \( \cos(\theta) = \cos( {-\theta} ) \), there are two possible solutions to the arccos function.

Inverse Kinematics Two Link System

In humanoid walking, the knee is always bending forward, so we pick the solid solution.

The dashed solution is also possible, but would be appropriate for birds. Note that birds' knees do not bend backwards - what you think is the knees are actually the heels.

Sine Law

Since we formed a right angle triangle, we now know

\[ l5 = \sin(\alpha_2) * l2 \]

But at the same time, we also know

\[ l5 = \sin(\alpha_1) * l3 \]

Applying the same reasoning to the other right angle triangle, we can also derive

\[ l4 = \sin(\alpha_1) * l1 \]

Sine Law

\[ \frac{\sin(\alpha_2)}{l3} = \frac{\sin(\alpha_1)}{l2} = \frac{\sin(\alpha_3)}{l1} \]

Inverse Kinematics Two Link System

Law of Sines

\[ \frac{l_3}{sin(\alpha_2)} = \frac{l_2}{sin(\alpha_1)} = \frac{l_1}{sin(\alpha_3)} \]

We can then calculate the angle \(α_1\)

# Inverse kinematic procedure
# Input: tx, ty, l1, l2

tx = -0.2
ty = 0.4

l1 = 0.5
l2 = 0.4

# theta1 = 50/180.0 * math.pi
# theta2 = 80/180.0 * math.pi

# 1. Calculate l3
l3 = math.sqrt( tx**2 + ty**2 )

# 2. Calculate α_2 knee angle
a2 = math.acos( (l1**2 + l2**2 - l3**2)/(2*l1*l2) )
print('a2', "{0:5.2f}".format(a2/math.pi*180.0) )

# 3. Calculate theta_2
theta2d = math.pi - a2
print("theta2", "{0:5.2f}".format(theta2d/math.pi*180.0) )

# 4. Calculate α_1
# Sine law: math.sin(a2)/l3 = math.sin(a1)/l2

a1 = math.asin( math.sin(a2)* l2/l3 )
print( "a1", "{0:5.2f}".format(a1/math.pi * 180.0) )
alpha = math.atan2( ty, tx )
print( "alpha", "{0:5.2f}".format(alpha/math.pi * 180.0) )

# Calculate theta1 from a2 and beta
theta1 = alpha - a1
print( f'theta1: {theta1/math.pi * 180.0:5.2f}')

Extension to Three Links

Humanoid robots usually have three links in the lateral plane (ankle, knee, hip)

In this case, we have an infinite number of solutions

Must add additional constraints

For example, the leg of a humanoid robot we commonly enforce

  • ankle must be 90 degrees to the ground
  • feet a parallel to the ground

Inverse Kinematics for More Links

  • Two link systems are the maximum that we can find a closed form solution for
  • If we have more links, then we can
    • decouple the kinematics (frontal plane, lateral plane, ...)
    • apply additional constraints (e.g., foot always parallel to the ground)
